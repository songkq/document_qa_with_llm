本项目用于文档问答，使用向量嵌入 + ES 做召回，再使用LLM做文档问答。

项目的整体架构如图：

Web框架使用Flask。

### 启动方式

1. 安装ElasticSearch（版本7.17.0）和Milvus（版本2.2.1），建议使用Docker搭建环境
2. 安装Python第三方模块，参考`requirements.txt`
3. 启动web服务: `python3 server.py`
